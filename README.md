# 🚀 Fraud Detection & Risk Analysis Dashboard

## 📊 Project Overview

This project analyzes over **100,000 financial transactions** to identify fraud patterns, detection gaps, and high-risk behaviors using **SQL and Tableau**.

---

## 🎯 Key Insights

* 🔴 Fraud Rate: **0.12%** across total transactions
* ⚠️ 100% fraud concentrated in **TRANSFER** and **CASH_OUT** transactions
* 💰 High-value frauds with amounts up to **₹1 Crore**
* 🚨 Fraud detection system showed **100% miss rate**
* 🔍 Majority of fraud cases resulted in **complete account drain**

---

## 🛠 Tools Used

* SQL (MySQL)
* Tableau (Dashboard + Story)
* Data Cleaning & Aggregation

---

## 📈 Dashboard Features

* KPI Summary (Fraud Rate, Total Fraud, Detection Failure)
* Fraud by Transaction Type
* Account Drain Pattern Analysis
* Fraud Trend Over Time
* Top Fraud Transactions

---

## 📚 Business Impact

This dashboard helps financial institutions:

* Identify high-risk transaction channels
* Detect fraud patterns early
* Improve fraud detection systems
* Prevent complete financial loss
