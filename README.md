
# 💳 Financial Transaction Fraud Detection

## 📌 Project Overview
Analysis of 1,00,050 financial transactions using SQL and Tableau 
to uncover fraud patterns, detection gaps, and high-risk behaviors.

---

## 🎯 Key Insights
- 🔴 Fraud Rate: **0.12%** across all transactions
- ⚠️ **100% of fraud** concentrated in TRANSFER & CASH_OUT only
- 💰 Highest single fraud: **₹1 Crore**
- 🚨 Bank detection system: **100% miss rate**
- 🔍 **96-100%** of fraud victims had accounts completely drained

## 📊 Dashboard Features
- KPI Summary — Fraud Rate, Total Fraud, Detection Failure
- Fraud by Transaction Type
- Top 10 Highest Fraud Transactions
- Fraud vs Legitimate Analysis
- Bank Detection Failure Analysis
- Account Drain Pattern Analysis
- Average Fraud Amount by Type

---

## 📚 Dataset
- **Source:** PaySim Synthetic Financial Dataset
- **Platform:** Kaggle
- **Records:** 1,00,050 transactions
- **Columns:** step, type, amount, nameOrig, nameDest, 
  oldbalanceOrg, newbalanceOrig, isFraud, isFlaggedFraud
